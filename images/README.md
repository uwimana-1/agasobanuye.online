# Images Folder

This folder contains movie poster images for the Agasobanuye streaming website.

## How to add images:

1. Place your movie poster images in this folder
2. Update the `poster` paths in `movies.js` to reference local images
3. Recommended image size: 300x450 pixels for posters
4. Supported formats: JPG, PNG, WebP
